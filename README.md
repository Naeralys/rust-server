# Client/Server fack-around project in Rust

This project is intended to explore basic network and file i/o for the Rust language.

### Installation
Assuming you have Rust installed, just install the dependencies with cargo.

### Running the service
One parameter with two choices can be supplied to `cargo run`, depending on the service you want to run: Client or Server.

Starting the server:
```cargo run -- server```

Starting the client:
```cargo run -- client```
